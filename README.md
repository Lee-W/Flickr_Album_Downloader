# Flickr Album Downloader
Download all the image in flickr albums of a given url

# Requirements
beautifulsoup4, requests
```sh
pip3 install -r requirements.txt
```

# USAGE
```sh
python3 Flickr_Album_Downloader.py
```
Input url as prompt ask.
